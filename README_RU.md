Плагин для [Claude Code](https://docs.anthropic.com/en/docs/claude-code) и Codex, который заставляет агента говорить как пещерного человека — сокращая **~75% токенов вывода** при сохранении полной технической точности. Теперь с [режимом 文言文](#文言文-wenyan-mode), [лаконичными коммитами](#caveman-commit), [однострочными ревью кода](#caveman-review) и [инструментом сжатия](#caveman-compress), который сокращает **~46% входных токенов** в каждой сессии.

Основано на вирусном наблюдении, что «язык пещерного человека» резко сокращает использование токенов LLM без потери технической сути. Мы сделали установку одной командой.

**Тот же ответ. Вы решаете, сколько слов.**

```
┌─────────────────────────────────────┐
│  СЭКОНОМЛЕНО ТОКЕНОВ   ████████ 75% │
│  ТЕХНИЧЕСКАЯ ТОЧНОСТЬ  ████████ 100%│
│  ПРИРОСТ СКОРОСТИ      ████████ ~3x │
│  ВАЙБЫ                 ████████ OOG │
└─────────────────────────────────────┘
```

- **Быстрый ответ** — меньше токенов генерировать = скорость летит
- **Легко читать** — никакой стены текста, только суть
- **Та же точность** — вся техническая инфа на месте, убрана только «вода» ([наука подтверждает](https://arxiv.org/abs/2604.00025))
- **Экономия денег** — на ~71% меньше токенов вывода = меньше затрат
- **Весело** — каждое ревью кода превращается в комедию

## Установка

Выберите вашего агента. Одна команда. Готово.

| Агент | Установка |
|-------|---------|
| **Claude Code** | `claude plugin marketplace add JuliusBrussee/caveman && claude plugin install caveman@caveman` |
| **Codex** | Склонируйте репо → `/plugins` → Найдите "Caveman" → Установить |
| **Gemini CLI** | `gemini extensions install https://github.com/JuliusBrussee/caveman` |
| **Cursor** | `npx skills add JuliusBrussee/caveman -a cursor` |
| **Windsurf** | `npx skills add JuliusBrussee/caveman -a windsurf` |
| **Copilot** | `npx skills add JuliusBrussee/caveman -a github-copilot` |
| **Cline** | `npx skills add JuliusBrussee/caveman -a cline` |

Установите один раз. Используйте в каждой сессии. Один камень. И всё.

### Что вы получаете

Автоактивация встроена для Claude Code, Gemini CLI и локальной настройки Codex. `npx skills add` устанавливает навык для других агентов, но **не** устанавливает файлы правил репозитория, поэтому Caveman не запустится автоматически без добавления фрагмента "Хотите всегда включено?" ниже.

> [!NOTE]
> Автоактивация работает по-разному для каждого агента. Подробности в основном README.md.

## Использование

Запуск с помощью:
- `/caveman` или `$caveman`
- "говори как пещерный человек"
- "режим пещерного человека"
- "меньше токенов, пожалуйста"

Остановка: "stop caveman" или "обычный режим"

### Уровни интенсивности

| Уровень | Триггер | Что делает |
|-------|---------|------------|
| **Lite** | `/caveman lite` | Убирает лишнее, сохраняет грамматику. Профессионально, но коротко |
| **Full** | `/caveman full` | Пещерный человек по умолчанию. Без артиклей, фрагменты, полный рык |
| **Ultra** | `/caveman ultra` | Максимальное сжатие. Телеграфный стиль. Сокращает всё |
