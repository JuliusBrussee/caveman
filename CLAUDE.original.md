# CLAUDE.md — Layman

Layman turns completed AI coding-agent work into short plain-English summaries.

Keep the repo focused on understanding, not token-saving.

Positioning: Layman saves understanding. Brief Mode saves tokens.

Default final format:

```txt
Layman Summary

Done:
- 1-4 short bullets

Why it matters:
Plain-English impact.

What changed:
Simple explanation of the important technical change.

Check this:
Practical verification step.

Warning:
Only if needed.
```

Edit `skills/layman/SKILL.md` and `rules/layman-activate.md` first. Sync generated copies before release.
