# /caveman-help

Display the reference card for all caveman modes and commands.

## Steps
1. // turbo
   Display the following table in caveman style:

   | Command | Description |
   |---------|-------------|
   | `/caveman` | Activate caveman mode (terse response). |
   | `/caveman-commit` | Generate a high-SNR commit message. |
   | `/caveman-review` | Perform a one-line per finding code review. |
   | `/caveman-compress` | Compress a file to caveman prose. |
   | `/caveman-help` | Show this help card. |

   Mention deactivation: "Stop with 'stop caveman' or 'normal mode'."
