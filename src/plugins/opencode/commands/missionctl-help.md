Show the missionctl quick-reference card.

| Command | What |
|---|---|
| `/missionctl` | Activate at default level (full) |
| `/missionctl lite` | Light compression — ~30% tokens dropped |
| `/missionctl ultra` | Maximum compression |
| `/missionctl wenyan[-lite\|-ultra]` | Classical Chinese compression |
| `/missionctl off` | Deactivate |
| `/missionctl-commit` | Terse commit message |
| `/missionctl-review` | One-line review findings |
| `/missionctl-compress <file>` | Compress a Markdown file |
| `/missionctl-stats` | Lifetime token-savings |

Natural language also works: "turn on missionctl", "stop missionctl", "normal mode".

Code, commits, security warnings: missionctl drops out automatically.
