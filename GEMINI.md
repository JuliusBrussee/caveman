## Antigravity IDE Integration

This repository is compatible with Antigravity IDE.

### Rules (Extensions)
- `caveman`: Passive rule to enforce caveman communication.

### Workflows (Slash Commands)
- `/caveman`: Activate caveman mode.
- `/caveman-commit`: Generate high-SNR commit messages.
- `/caveman-review`: Perform ultra-compressed code reviews.
- `/caveman-compress`: Compress natural language files.
- `/caveman-help`: Show help card.
